# STM32_PracticeProject
基于 STM32 的双机通讯直流电机 PID 闭环控制系统 
本项目实现了一个由上位机和下位机组成的协同控制系统。上位机负责 ADC 信号采集（速度映射）、按键逻辑处理（非阻塞）及指令下发；下位机负责直流电机编码器测速、双环 PID 调控、Flash 数据持久化、OLED 实时监控以及 MPU6050 欧拉角姿态解算。
此项目完成的功能有：
上位机
1.adc采集电位器数值并映射，正常挡位为-100～+100，低速档为-50～+50
2.按键1一键启停电机，上电时电机处于停止状态
3.按键2切换速度挡位，在正常挡和低速档之间切换
4.通过USART与下位机通讯，将启停电机的状态，速度挡位以及设定的速度值传给下位机
下位机
1.编码器读取直流电机实际转动速度
2.通过pid调控使电机稳定在设置速度
3.通过flash保存并刷新按下按键1时从启动到停止读取的实际电机速度
4.通过iic接口0.96寸oled屏幕显示要求内容，包括1.电机启动状态（on/off）2.速度档位电机设置速度4.电机实际速度5.flash保存的速度值
5.使用mpu6050陀螺仪，将读取三轴数据转换成欧拉角（yaw、roll、pitch），显示在oled屏幕上
